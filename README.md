# Arx Fatalis Port

(Arx Libertis?)
Port to GCC / OpenGL / Unix

## Dependencies

Currently wine and libjpeg, and maybe some other headers like zlib.h.
Wine is only a temporary solution.

## Compile

`$ cmake .`

`$ make`

## Run
 
`$ wine arx.exe.so`
